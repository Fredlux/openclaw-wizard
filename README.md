# OpenClaw Setup Wizard Local

Version locale du setup wizard d'OpenClaw

## Démarrage

```bash
npm install
npm run dev
```

Le wizard est disponible sur `http://localhost:3000`

## Contenu

- `index.html` — page principale du wizard
- `vite.config.js` — configuration Vite
- `package.json` — dépendances
