🚀 OpenClaw Setup Wizard

Configuration locale guidée pour votre assistant IA

1️⃣ Prerequisites

Vérifiez que ces outils sont installés sur votre Mac :

  • Node.js 22+ ( node --version )
  • npm ( npm --version )
  • Homebrew ( brew --version )

Si Node.js n'est pas installé : brew install node

2️⃣ Install OpenClaw

Installez OpenClaw globalement :

npm install -g openclaw

Vérifiez l'installation :

openclaw --version

3️⃣ Configuration (Wizard)

Lancez l'assistant de configuration interactif :

openclaw onboard

Ou en mode non-interactif :

openclaw onboard --non-interactive --accept-risk --auth-choice token --token-provider anthropic --token "$ANTHROPIC_API_KEY" --gateway-bind loopback --gateway-auth token --gateway-token "$(openssl rand -hex 32)" --skip-channels --skip-skills --skip-ui

4️⃣ Connecter un Canal

WhatsApp (recommandé) :

openclaw channels login --channel whatsapp --verbose

Scannez le QR code avec votre téléphone.

5️⃣ Démarrer le Gateway

openclaw gateway start

Vérifiez le statut :

openclaw health
